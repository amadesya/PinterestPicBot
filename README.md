## ЗАПУСК 

1. Установка библиотека
```
pip install aiogram playwright beautifulsoup4 aiohttp && playwright install

playwright install
```

2. В файле .env подставить значение своего токена
`
TOKEN = ВАШ_ТОКЕН
`
3. Запустить бота
